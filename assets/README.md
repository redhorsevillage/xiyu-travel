# assets/

本網站的所有視覺（景緻背景、Shisa／石敢當圖示、明信片插畫、波浪動畫）皆以
**CSS 漸層 + 內嵌 SVG + Canvas 繪製**完成，無需任何外部圖檔即可運作。

若日後要替換為實景照片，建議規格如下，放入本資料夾後於 `css/style.css`
與 `js/main.js` 的 `SCENES` 物件改用 `url('../assets/xxx.jpg')`：

| 檔名建議            | 對應景點                 | 比例 / 尺寸        |
|--------------------|--------------------------|--------------------|
| `erkan.jpg`        | 二崁聚落（赤瓦老屋）     | 直式 3:4，≥1200px  |
| `basalt.jpg`       | 大菓葉柱狀玄武岩         | 橫式 4:3，≥1200px  |
| `lighthouse.jpg`   | 漁翁島燈塔               | 橫式 4:3，≥1200px  |
| `seawall.jpg`      | 內垵海堤・西嶼海岸       | 橫式 4:3，≥1200px  |

照片請使用自有或具授權之素材（CC0 / 已取得使用許可）。

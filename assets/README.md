# assets/

本網站（v4「古樸漁港」）的視覺以三種方式構成：

1. **CSS 漸層 + 內嵌 SVG + Canvas 繪製** —— 介面裝飾、圖示、明信片貼紙、波浪與背景後備底圖，皆由程式生成，無需外部圖檔。
2. **真實實景照片（Wikimedia Commons）** —— 「海天奇景」12 個景點與部分「鹹鮮風味」美食，於 JavaScript 中以圖片網址嵌入；載入失敗時自動回退為 CSS 漸層底圖。照片授權見根目錄 [`NOTICE.md`](../NOTICE.md)。
3. **Google Maps 內嵌地圖** —— 「探活踩風」行程規劃以 iframe 嵌入，並依路線動態產生多點導航連結。

> 因此本資料夾目前**不需放置任何圖檔**即可完整運作。

## 若要改用本地圖檔

把圖片放入本資料夾後，於 [`js/main.js`](../js/main.js) 的 `WIKI`（景點）或 `FOODS[].img`（美食）改為相對路徑即可，例如：

```js
erkan: 'assets/erkan.jpg',
```

景點底圖建議橫式 16:9 或 4:3、寬度 ≥ 1200px；美食建議方形或 4:3。
請務必使用自有或具公開授權（CC0 / CC BY / CC BY-SA 等）之素材，並於 `NOTICE.md` 標註來源與授權。
